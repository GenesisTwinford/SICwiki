# 要件定義

## 概要
プログラミング等の複雑な事象を、体系的かつ直感的に理解するためのwiki。  
あるユーザーがコースを開発し、別のユーザーがそれを受講。

## MVP機能
- ログイン
- ページ閲覧
- ページ作成
- ページ編集
- 編集履歴閲覧
- ログインユーザーのみ編集可能に

## 追加予定の機能
- 体系化によって、自身の学習の位置を把握できるUI。
- 説明について、  
上級者向け⇔初心者向け。  
わかりやすい⇔わかりにくい。  
の評価を基に、記事を採用。
- 説明の挫折者数を表示可能に。
- 説明の直後に問題集。
- デフォルトの説明の難易度を設定可能に。
- しおりの追加。
- 読了済み（理解済み）機能の追加。
- 説明の特定の箇所に補足と質問を可能に。
- 説明の追加・誤字修正・書いたコメントの評価を得点源としたユーザーランクの追加。stackoverflowを参照。
- マイページに学習の統計追加。
- AIに質問可能に。
- AIの無料枠が余ったら記事を追加。

## 非機能要件
- パフォーマンステストを実施（Chrome開発者機能）
- セキュリティテストを実施
