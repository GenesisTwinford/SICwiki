# 要件定義

## 概要
本プロダクトは、プログラミングといった理系の複雑な学問を、初学者・独学者が、学習内容の全体構造と現在地を把握しながら学べるコース型学習サービスである。
Wikiのような形式で複数人の有志が教材を作成。
Duolingoのような形式で受講者のモチベーションを維持。ブログのような形式で受講者は理解を深めるが、コメントによる補足等で差別化。

## MVP機能
- ログイン
- ログアウト
- 編集履歴閲覧

## 追加予定の機能
- 体系化によって、自身の学習の位置を把握できるUI。
- URL：/
  - 左にロゴ、通知、プロフィールを配置。
  - 中央にコース一覧を配置。
  - 右にユーザー統計を配置。
    - デイリータスクの進捗
    - FF内での順位
- URL：/article/~  
  - ページ閲覧（ログイン不要）。
  - ページ作成。
  - 記事タイトル・概要・セクション１・問題集・セクション２・問題集......の流れ。
  - セクション内の文章単位で、ユーザーによる補足コメント、挫折箇所へのリアクションを可能に。
  - セクション単位で、記事の編集を可能にする。  
  上級者向け⇔初心者向け、わかりやすい⇔わかりにくい  
  のユーザー評価を可能とし、それを基に記事を採用。
  - デフォルトの説明の難易度を設定可能に。
  - しおり、読了（理解）済み、説明の難易度。
学習した単元の数、学習時間など。
- 説明の追加・誤字修正・書いたコメントの評価を得点源としたリーグ制度の追加。Duolingo、stackoverflowを参照。
- 最終テストをクリアした際のバッジの追加
- マイページに学習の統計追加。

## 非機能要件
- パフォーマンステストを実施（Chrome開発者機能）
- セキュリティテストを実施
